# QUIZ-APP
This Quiz App is an interactive application that allows users to test their knowledge through multiple-choice questions. It displays questions sequentially, tracks user responses, calculates the final score, and shows results with a clean UI and smooth user experience.
# Quiz App üìù

A Django-based Quiz Application that allows users to answer multiple-choice questions, track responses, calculate scores, and view results at the end.

## Features
- Multiple-choice questions
- Real-time score calculation
- Clean and user-friendly UI
- Result summary at the end

## Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite

## How to Run
1. Clone the repository
2. Install dependencies  
   `pip install -r requirements.txt`
3. Run the server  
   `python manage.py runserver`
4. Open `http://127.0.0.1:8000/`
